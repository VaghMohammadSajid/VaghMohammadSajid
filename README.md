## 🧭 How to run (example)
```bash
# clone project
git clone <repo-url>
cd <repo-folder>

# create & activate venv
python -m venv env
source env/bin/activate     # macOS / Linux
# .\env\Scripts\activate    # Windows

# install & run
pip install -r requirements.txt
cp .env.example .env        # set DB creds
python manage.py migrate
python manage.py runserver
